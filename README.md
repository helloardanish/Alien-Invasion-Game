# Alien Invasion

Alien invasion game project to learn python3.

#### Requirements:
 - python3
 - pip
 - pygame library (pip3 install pygame)

#### Playing:
 - arrows and space(fire bullet)

#### Run it
    $ python3 alien_invasion.py
